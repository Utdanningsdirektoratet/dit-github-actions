# DIT - GitHub Actions

Shared GitHub Actions for CI/CD automation in DIT. Contains reusable actions for build, deploy and maintenance.

## Usage

Reference actions with `{owner}/{repo}/{path}@{ref}`:
```yaml
- uses: Utdanningsdirektoratet/dit-github-actions/docker/build@v1
```
